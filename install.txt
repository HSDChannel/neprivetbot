RU: Как работать с ботом?
Для начала, надо установить Python: https://www.python.org/
Далее установить PyTelegramBotAPI с помощью команды: pip install pyTelegramBotAPI
Теперь надо получить токен
Открываем телеграм и вводим в поисковике BotFather
Далее вводим команду /newbot
Теперь надо ввести имя вашего бота
После этого вводим имя бота на латинице и добавляем в конце "bot" (без кавычек)
Должно получится примерно так: imahhbotabot
Далее копируем токен
Теперь открываем main.py в блокноте
Вместо "your_token" заменяет на токен, которые мы взяли из бота
Сохраняем и запускаем!
